Markdown語法規則

本文詳細列舉了Hugo文件中的markdown語法規則，學會了就能愉快地開始博客寫作了。

標題

如下六個 HTML <h1>—<h6> 元素代表了六個級別的章節標題。 <h1> 最高， <h6> 最低。

段落

他们只是坐在电视和电脑前，看着那里传播的各种信息和娱乐，到死都是如此，就这样漫无目的地生活。无论吃饭还是洗澡，工作还是恋爱，都只是去完成而已。他们毫无自觉地消磨时间，却还要感叹人生苦短。他们靠免费的娱乐度日，陪伴他们的是电视和网络。丰富的信息和单调的生活只能产生武断的想法和憎恨。——《魔王》伊坂倖太郎

引用

“引用”元素表示從另一個來源引用的內容，可選擇在 footer或cite 元素內引用，也可以和註釋和縮寫等內聯更改。

無注釋的引用

丰富的信息和单调的生活只能产生武断的想法和憎恨。 注意 在引用内部也可以使用 Markdown語法規則。

Blockquote with attribution

丰富的信息和单调的生活只能产生武断的想法和憎恨。

— 《魔王》伊坂倖太郎1

表格

表格不是核心 Markdown 規範的一部分，但 Hugo 支持該語法的使用。

用表格形式表示一下OR語法的布爾值：

OR TRUE FALSE
TRUE TRUE TRUE
FALSE TRUE FALSE

在表格中使用markdown語法

在表格中    使用    Markdown    語法
斜體 加粗 刪除綫    代碼

代碼塊

使用反引號的代碼塊 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>

縮進四個空格的代碼塊

（即按兩下Tab）

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>

帶有 Hugo 內部高亮簡碼的代碼塊 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>

插入圖片
