🍑 How to Use Emoji

  • ZH
  • EN

This article mainly introduces the use of Emoji in the Hugo framework, adding cute emoji to your blog posts.

Method to realize

Displaying Emoji in the article can directly call the emojify function and use the [Inline Shortcodes](https://gohugo.io/templates/shortcode-templates/#inline- shortcodes) function implementation.

p.s.: After opening InlineShortcodes, it will have many powerful functions, such as inserting YouTube videos, etc.

Here is just a chance to enjoy Jinyoung’s “Dive” with you~~

Detailed Method

If you need to use it globally, you need to put “config.toml, config.yaml, or config.json” in the root directory of the website (according to the specific situation, my file is config.toml, so the following code is also modified by the .toml file For example. For the modification method of other files, please refer to the content in the Inline Shortcodes link above) Add two lines of code to the file:

enableInlineShortcodes = true
enableEmoji = true

In this way, emoji can be used directly in the website. After entering the code corresponding to emoji, the effect is as follows:
