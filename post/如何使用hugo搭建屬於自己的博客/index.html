Install Hugo

MAC 

brew install hugo
hugo version

WIN

  1. Download corresponding .zip document in the hugo release page
  2. unzip hugo.exe file and put it into D:\Software\hugo(You can choose your own path)
  3. add the above into path
    • Right click my computer
    • Attributes
    • Advanced system settings
    • environment variables
    • double click path
    • new path
  4. Restart the terminal, run hugo version. The version number indicates success.

Build a Blog

Enter hugo official websiteï¼ˆhttps://gohugo.io/getting-started/quick-start/ï¼‰, follow the steps to build it step by step.

Noticeï¼š

hugo new site quickstart

The quickstart need to be changed into your github repository name and add -generator.

Create a New Terminal

  1. Run hugo in the terminal (generate the public folder); you can also use hugo -F --cleanDestinationDir (generate a new folder)
  2. Create a new .gitignore on the outermost side and enter:

/public/
which means ignore my public, it will become a new folder alone.

  1. cd public
  2. git init
  3. git add .
  4. git remote add origin git@github.com:[your user name]]/[your user name].github.io.git
  5. git branch -M main
  6. git push -u origin main

Create the Second Article

  1. Run hugo new posts/the Second Article.md in xxx.github.io-creator (Make sure you are not in public )
  2. Run code posts/the Second Article.md to edit the file. Make sure not the delete the original content and write in another line to input your content.
  3. Run hugo -D to get the new public
  4. Enter public for next step
    1. git add . (Don’t forget the .)
    2. git commit -m update
    3. git push -f (Force upload)
  5. Wait a few minutes for a second post to appear on your blog!

Upload Pic

Method1

  1. Open the sm.ms website, upload the picture, and get the url of the picture.
  2. Use ![](url) to show the pic in your article.

Method2

  1. Store the image file 1.png in the static directory and get static/1.png
  2. Write ![](/1.png) in the blog Markdown to see the image corresponding to static/1.png
  3. If you want to create an images directory in static, just create it directly, and add /images in front of the corresponding image, for example
    1. Put the image file 2.png in the static/images directory to get static/images/2.png
    2. Just write ![](/images/2.png) in the blog

In addition, you can also Google “Hugo insert picture” to find the answer.
