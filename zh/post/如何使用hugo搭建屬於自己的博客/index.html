🌴 如何使用Hugo搭建屬於自己的博客

  • ZH
  • EN

Hugo用Go語言写成，可以幫助我們快速搭建一個優雅的網站。

Hugo安裝

MAC 

brew install hugo
hugo version

WIN

  1. hugo release頁面下載對應版本zip文件
  2. 解壓将hugo.exe文件放入D:\Software\hugo（路徑可自選，最好是是英文）
  3. 将以上路徑加入path
    • 右鍵我的電腦
    • 屬性
    • 高級系統設置
    • 環境變量
    • 雙擊path
    • 新增路徑
  4. 重啓終端，運行hugo version，看到版本號就説明成功了。

搭建博客

进入hugo官网（https://gohugo.io/getting-started/quick-start/），按照該步驟一步步進行搭建。

注意：

hugo new site quickstart

其中quickstart需要改成github仓库名加“-generator”。

新建一个終端

  1. 在終端中運行hugo（生成public文件夾）；也可以用hugo -F –cleanDestinationDir（生成全新的文件夾）
  2. 在最外面新建.gitignore，其中輸入：

/public/
即不要管我的public，它要自成一个文件夾。

  1. cd public
  2. git init
  3. git add .
  4. git remote add origin git@github.com:你的用戶名/你的用戶名.github.io.git
  5. git branch -M main
  6. git push -u origin main

创建第二篇博客

  1. 在 xxx.github.io-creator 目錄（注意確保自己不在 public 目錄）里運行 hugo new posts/第二篇博客.md
  2. 運行 code posts/第二篇博客.md 对文件進行編輯，注意不要把文件原本的内容给刪了，直接在後面另起一行寫新内容。
  3. 運行 hugo -D，得到新的 public 目錄
  4. 進入 public 目錄，執行一下操作
    1. git add . 注意有一个点
    2. git commit -m update
    3. git push -f 其中 -f 是强制上傳的意思
  5. 等待几分鐘后，你的博客就会出现第二篇文章了！

上传圖片

方法1

  1. 打開 sm.ms 網站，將圖片上傳上去，得到圖片的 url。
  2. 在文章里使用 ![](url) 語法即可顯示圖片

方法2

  1. 將圖片文件 1.png 存放到 static 目錄里，得到 static/1.png
  2. 在博客 Markdown 裏寫 ![](/1.png) 就能看到 static/1.png 對應的圖片了
  3. 如果你想在 static 裏創建 images 目錄，直接創建就好，對應的圖片前面加上 /images 即可，比如
    1. 將圖片文件 2.png 放到 static/images 目錄裏，得到 static/images/2.png
    2. 在博客裏寫 ![](/images/2.png)即可

另外，你还可以Google一下「Hugo 插入圖片」找找答案。
